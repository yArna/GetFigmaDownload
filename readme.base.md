## nupkg 文件
Window 安装包是 `.nupkg` 格式的，其实是 zip 压缩包，你可以添加 `.zip` 后缀，然后用压缩工具解压，在其中的 `/lib/net45` 文件夹下就是 Figma 程序了，你可以把它放到任何目录下运行。
